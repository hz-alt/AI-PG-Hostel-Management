===========================================
AI PG & Hostel Management System
===========================================

Project Type:
Frontend Prototype (AI using MERN Internship)

-------------------------------------------
Project Description
-------------------------------------------

AI PG & Hostel Management System is a smart hostel management application developed using HTML, CSS, and JavaScript. It helps manage student records, room allocation, room availability, and provides AI-based room recommendations through a simple and user-friendly interface.

This project is created as a frontend prototype for the AI using MERN Internship.

-------------------------------------------
Features
-------------------------------------------

✔ Dashboard with Hostel Statistics
✔ Add Student
✔ Room Allocation
✔ Student Search
✔ AI Room Recommendation
✔ Occupancy Progress Bar
✔ Live Date & Time
✔ Dark Mode
✔ Room Status Cards
✔ Notice Board
✔ Emergency Contact Section
✔ Responsive User Interface

-------------------------------------------
Technology Used
-------------------------------------------

Frontend:
- HTML5
- CSS3
- JavaScript

Future MERN Stack:
- MongoDB
- Express.js
- React.js
- Node.js

-------------------------------------------
AI Functionality
-------------------------------------------

The AI Recommendation module suggests the best available room based on:
- Room Availability
- Occupancy Status
- Student Comfort
- Room Facilities

(Currently implemented as a frontend demonstration using JavaScript logic.)
